# hackerearth_carnival_wars

## Best model is the best saved model which gives the highest accuracy on test.csv

Requirements.txt contains all the necessary package with its versions used.

### Getting Started

    $ pip install -r requirements.txt
    
### Python 3.7.1
    
    $ python Carnival_wars_train_model.py
    
This will train our model.

    $ python Result_prediction.py
    
This will give output for test dataset and save the output in csv file.

    Jupyter notebooks are also attached for visualisation and easy understanding of code.
